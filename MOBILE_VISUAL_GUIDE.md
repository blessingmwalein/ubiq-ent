## ğŸ“ Responsive Sizing Reference

### Typography
- **Headings:**
  - Mobile: `text-3xl` (30px)
  - Tablet: `text-4xl-5xl` (36-48px)
  - Desktop: `text-7xl` (72px)

- **Body Text:**
  - Mobile: `text-base` (16px)
  - Tablet: `text-lg` (18px)
  - Desktop: `text-xl` (20px)

### Spacing
- **Padding:**
  - Mobile: `p-4` (16px)
  - Tablet: `p-6` (24px)
  - Desktop: `p-8` (32px)

- **Gaps:**
  - Mobile: `gap-2` (8px)
  - Tablet: `gap-3` (12px)
  - Desktop: `gap-4` (16px)

### Components
- **Buttons:**
  - Mobile: `h-12` (48px) - Full width
  - Tablet: `h-12` (48px) - Auto width
  - Desktop: `h-14` (56px) - Auto width

- **Inputs:**
  - Mobile: `h-12` (48px)
  - Tablet: `h-12` (48px)
  - Desktop: `h-14` (56px)

- **Touch Targets:**
  - Minimum: `44x44px` on all touch devices
